# Android touch sample rate test tool

This tool can help you check the touch sample rate of the Android device.

## Supported input method:
* Touch
* Mouse
* Stylus (S-Pen)

# Test result
- Galaxy Tab S7+
    - Touch: 120 Hz
    - S-Pen (Hover): ~333 Hz
    - S-Pen (Down): ~370 Hz

- Galaxy S20 FE
    - Touch: 120 Hz

- Dell S2240T Monitor
    - Touch: 200Hz
